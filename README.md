# Platoon E2E Test Suite

Automated end-to-end tests for the Platoon payroll platform.

## Tech Stack

- Cypress
- JavaScript

## Test Coverage

- Landing page & navigation
- Account creation
- Authentication (login/logout)
- KYC document upload
- Wallet funding
- CSV payroll upload
- Employee data import
- Payroll scheduling & submission
- Payslip download

## How to Run

npm install
npx cypress open
